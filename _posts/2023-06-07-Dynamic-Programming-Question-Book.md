# 동적 계획법이란?
동적 계획법(Dynamic Programming)이란 표 만들기(Memoization)를 통해서 문제를 해결해 나가는 방식입니다.

중복된 계산을 하지 않아도 되니 시간 복잡도를 굉장히 줄일 수 있습니다.

# 동적 계획법 문제를 풀 때 생각해야 할 것들
1. 기저 사례
2. 이 항과 그 전 항과의 관계 (점화식 세우기)
3. 어떠한 방식을 사용할 것인지
- Pull DP / Push DP
- Top-Down / Bottom-Up
